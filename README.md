# Sprint 2 SQL · Bases de dades relacionals

## Descripció del projecte
Aquest sprint té com a objectiu repassar les nocions bàsiques de les bases de dades relacionals mitjançant l’anàlisi d’una base de dades d’una empresa de venda online.  
S’han utilitzat consultes SQL per treballar amb transaccions i informació corporativa de les empreses participants.

---

## Contingut
- `Sprint_2_informe.pdf` → explicació i resultats

- `arxius csv/` → dades per a la importació
- `arxius sql/` → estructura i dades inicials i dades d'importació
